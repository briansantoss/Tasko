SOBRE NÓS

Este repositório foi desenvolvido para hospedar a aplicação web Tasko, criada para o projeto WebPython da disciplina Tópicos Especiais em Software. Tasko é um sistema de gerenciamento de tarefas que visa fornecer uma interface intuitiva e eficiente para os usuários gerenciarem suas tarefas. Utilizando a estrutura Django, a aplicação aproveita o princípio Do Not Repeat (DRY) para promover desenvolvimento ágil e código organizado. Django, com sua abordagem de “baterias incluídas”, fornece recursos básicos como autenticação de usuário, um sistema ORM para gerenciamento de banco de dados e um painel de administração configurável. No Tasko, cada tarefa consiste em um título, descrição e status, sendo o status utilizado para indicar o andamento da tarefa como “Em Andamento” ou “Concluído”. Essa estrutura simplificada facilita a organização e o controle de tarefas. O objetivo do projeto é demonstrar como o Django pode ser aplicado para desenvolver uma aplicação web robusta e escalável, proporcionando uma visão clara dos benefícios e funcionalidades do framework no contexto da gestão do trabalho.

MAIS INFORMAÇÕES

A aplicação Tasko também oferece uma série de funcionalidades adicionais para melhorar a experiência do usuário. Além da gestão básica de tarefas, o sistema permite a criação de categorias para organizar as tarefas de forma mais eficiente. Também é possível definir prioridades para cada tarefa, o que ajuda a identificar as mais urgentes. O sistema de notificações mantém os usuários informados sobre prazos e alterações importantes. Além disso, a aplicação é projetada para ser responsiva, garantindo que os usuários possam acessar e gerenciar suas tarefas em qualquer dispositivo, seja um computador, tablet ou smartphone. Com essas funcionalidades, o Tasko visa se tornar uma ferramenta essencial para a gestão eficaz do tempo e das tarefas diárias.
